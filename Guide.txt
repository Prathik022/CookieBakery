Guide to run the server:

- go to the backend folde
- open the server.js file and change the url for the mongodb(give your database password and url)
- use command { npm init}
- download all the necessary packages { npm i axios mongoose cors express nodemon}
- config the nodemon in package.json
- { npm start}
